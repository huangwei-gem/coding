## 快速开始

1. 双击 `index.html` 在浏览器中打开
2. 在文本框中粘贴或输入要检测的内容
3. 点击 **「开始检测」**（或按 `Ctrl + Enter`）
4. 查看标红高亮的违规内容、违规详情和修改建议

### 系统要求

- 浏览器：Chrome 80+ / Edge 80+ / Firefox 80+ / Safari 14+
- OCR 功能需要联网（加载 Tesseract.js）
- ASR 语音识别仅支持 **Chrome / Edge**（需要 HTTPS 或 localhost）

## 文本检测

### 三层风控引擎

系统默认启用全部三层检测，可在输入框下方独立开关：

- **规则引擎** — 精确匹配 200+ 内置违禁词，分政治、师德师风、广告营销、低俗违规、教学合规五大类
- **NLP 增强** — 识别谐音字（收fei→收费）、形近字、拼音替代、拆分插字（收 费）、英文替代（vx→微信）、数字谐音（520→我爱你）
- **语义分析** — 检测上下文组合暗示，如"周末来我家补课"（时间+地点+行为组合）、"最近手头紧"（索礼暗示）、"全班就你不会"（公开羞辱）

### 检测结果

- **原文标注**：违规内容标红高亮，鼠标悬停查看违规类型和严重程度
- **违规详情**：列出每条违规的词汇、上下文、分类和严重程度
- **修改建议**：根据违规类型自动生成修改建议

### 快捷操作

- 加载示例文本（含多种违规类型演示）
- 导入 TXT 文件自动检测
- 清空输入内容

## 图片/语音识别

### OCR 图片文字识别

1. 点击上传区域或拖拽图片到虚线框
2. 系统自动调用 Tesseract.js 进行 OCR 识别（支持中文+英文）
3. 识别结果自动填充到文本框
4. 点击 **「检测识别结果」** 送入风控引擎检测

支持格式：JPG、PNG、BMP、WebP

> 注意：首次加载需要从 CDN 下载 Tesseract.js 语言包，请保持网络通畅。

### ASR 语音转文字

1. 点击麦克风按钮开始语音识别（Chrome/Edge 浏览器）
2. 对着麦克风说话，语音实时转写为文字
3. 再次点击麦克风停止识别
4. 点击 **「检测语音内容」** 送入风控引擎检测

> 注意：语音识别仅在支持 Web Speech API 的浏览器（Chrome/Edge）上可用。
> 识别结果支持手动编辑修正。

## 违禁词库管理

### 系统词库

内置 200+ 违禁词，分 5 大类别 + 自定义类别：

| 类别 | 示例 |
|------|------|
| 政治类 | 法轮功、台独、分裂国家、邪教、歪曲历史 |
| 师德师风类 | 真笨、废物、发红包、有偿补课、加微信私聊、做我女朋友 |
| 广告营销类 | 购买课程、限时优惠、扫码领取、售卖资料 |
| 低俗违规类 | 傻逼、色情、赌博、迷信、吸毒、卧槽 |
| 教学合规类 | 保证提分、包过、超前教学、押题、竞赛包过 |
| 自定义 | 用户自行添加 |

**类别开关**：点击类别标签可单独启用/关闭某类检测。

### 自定义词库

- **添加**：输入词汇，选择分类和严重程度，点击添加
- **批量导入**：支持 `词汇|分类|严重程度` 格式批量导入
- **导出**：导出自定义词库为文本文件
- **清空**：一键清空所有自定义词

### 白名单（豁免词）

- 加入白名单的词汇在任何类别中都不会被判定为违规
- 用于避免教学专用术语、学校名称、人名地名等被误判
- 申诉通过后自动加入白名单

## 人工复核中心

### 复核流程

1. 检测完成后点击 **「提交复核」** 将检测结果送入复核队列
2. 管理员在复核标签页查看待复核列表
3. 点击 **「复核」** 查看详情，可选择：
   - **确认违规** — 标记为确认违规
   - **判定误判** — 标记为误判
   - **标记疑似** — 标记为疑似待进一步判断
4. 复核备注可填写处理意见

### 申诉流程

1. 教师检测后点击 **「申诉误判」**
2. 填写申诉原因，说明为何认为是误判
3. 管理员在申诉管理列表中处理：
   - **通过** — 标记为误判，对应词汇自动加入白名单
   - **驳回** — 维持违规判定

### 复核报告

支持导出复核 CSV 报告，包含 ID、时间、内容、违规数、状态、备注。

## 常见问题

### Q: 数据保存在哪里？会丢失吗？
所有数据保存在浏览器本地存储（localStorage），清除浏览器缓存会丢失数据。建议定期导出备份。

### Q: OCR 识别不支持/识别率低？
Tesseract.js 识别率受图片质量影响，建议上传清晰的扫描件或截图。首次使用需要加载语言包，请保持网络连接。

### Q: 语音识别不支持？
ASR 功能需要 Chrome 或 Edge 浏览器，且需要 HTTPS 或 localhost 环境。在部分浏览器中可能因权限限制不可用。

### Q: 如何备份数据？
自定义词库可通过导出功能保存为文本文件。检测历史可通过 CSV 导出。复核报告也可导出。

### Q: 误判怎么办？
将误判词汇加入白名单，或通过申诉流程由管理员处理。申诉通过的词汇会自动加入白名单。

### Q: 词库如何更新？
系统词库为静态数据，如需更新请通过自定义词库添加。建议定期关注教育部、网信办发布的最新师德规范和管控要求。
