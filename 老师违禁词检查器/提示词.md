# 老师违禁词筛选器产品方案（AI 产品经理视角）

整体思路：**先定场景 & 规则→设计 AI 能力→搭建产品架构→落地流程 & 迭代→风险管控**，全程贴合教培 / 校内课堂、直播授课、课件审核、师生沟通等真实教学场景。

## 一、前期调研 & 需求定义（第一步：明确做什么、给谁用）

### 1. 目标用户 & 使用场景

按使用角色拆分，区分核心场景：

1. **校方 / 教务管理员**：批量审核课件、教案、班级群文字、课后作业评语、公告
2. **授课老师**：实时自查课堂话术、直播口播、私信沟通内容
3. **平台运营（线上教培）**：直播课堂实时风控、录播内容回查、评论区管控

内容形态：**文本（文字 / 聊天 / 课件）、语音转文字（课堂口播、录音）、图片文字（板书、PPT 截图）**

### 2. 违禁词分类（核心规则底座，区分教学行业专属违禁）

结合教育行业合规要求、师德规范、网络风控，分 5 大类，同时区分**严拦截、预警提醒、人工复核**三个管控等级：

1. **政治类**：敏感时政、不当言论、错误价值观、违规政治表述（最高优先级，直接拦截）

2. **师德师风类（教育专属）**

   - 体罚、辱骂、嘲讽学生、人身攻击、歧视话术
   - 索要 / 收受礼品、红包、补课收费、有偿补课暗示、校外私课引流
   - 师生不当交往、暧昧话术、越界言论

   

3. **广告营销类**：校外机构引流、带货、售卖资料 / 课程、诱导付费

4. **低俗违规类**：脏话、色情、暴力、迷信、不良网络用语

5. **教学合规类**：违规补课宣传、超前教学、违规竞赛、虚假升学承诺

### 3. 核心功能目标

- 实时检测：直播、实时聊天**毫秒级预警**
- 批量检测：文档、录音、图片**批量一键审核**
- 分级处置：拦截 / 提醒 / 仅记录，不一刀切
- 可自定义：学校 / 机构能自主添加本校专属违禁词、豁免词
- 溯源留痕：审核日志、违规记录可查询、导出（合规备查）

## 二、AI 能力选型 & 技术方案（产品侧定义技术边界）

结合教育场景特点，采用**规则引擎 + 大模型 AI + 传统 NLP**融合方案（纯规则漏检高，纯大模型成本高、速度慢）

### 1. 技术架构组合（三层风控，层层过滤）

#### （1）第一层：基础规则引擎（高速拦截，兜底）

- 能力：精准匹配**固定违禁词、词组、短句**，支持通配符、前后缀匹配
- 适配场景：高频固定违禁词（脏话、明确违规话术、固定敏感词）
- 优势：响应极快、零误杀、占用资源低，用于实时直播 / 聊天

#### （2）第二层：传统 NLP 语义识别（解决变体、谐音、拆字）

针对人为规避行为：谐音、拼音、错别字、拆分字、表情包替代、缩写

- 能力：形近字、谐音字、异体字、语序打乱、短句语义匹配
- 举例：把 “收费” 写成 “收 fei”“收米”，系统识别语义并预警

#### （3）第三层：大语言模型 LLM（高阶语义理解，教育场景核心）

解决**长文本、上下文语义、隐含违规、委婉话术**（教育场景重灾区）

- 能力 1：上下文理解（单句没问题，整段暗示有偿补课 / 私下交易则判定违规）
- 能力 2：意图识别（区分正常沟通 vs 变相索要礼物、私下约课）
- 能力 3：情绪 & 语气识别（识别嘲讽、辱骂、冷暴力类话术）
- 能力 4：长文档解析（教案、万字课件逐段审核）

#### （4）多模态能力（覆盖语音、图片）

1. **语音转文字 ASR**：课堂录音、直播语音实时转文本，再送入风控模型，支持方言、课堂嘈杂环境降噪
2. **OCR 图文识别**：PPT、板书、截图、手写文字提取文本后审核

### 2. 模型优化方向（产品提需求给算法）

1. **行业微调**：用**教育课堂、师生沟通、教培话术**专属语料微调模型，降低通用模型的误判
2. **分级阈值配置**：管理员可调节风控严格度（幼儿园 / 中小学 / 高校可设置不同标准）
3. **豁免库**：设置白名单（正常教学词汇、学校专有名词、人名地名，避免误杀）

## 三、产品核心功能模块（落地功能清单，分端设计）

按**管理后台、教师端、实时风控中台**三大模块设计，区分 Web 端、小程序、客户端。

### 模块 1：管理员后台（核心配置 + 数据管理）

1. **违禁词库管理**

   - 系统默认词库（平台统一更新，同步国家 / 教育部门新规）
   - 自定义词库：学校 / 机构手动新增、编辑、删除违禁词，支持批量导入 Excel
   - 白名单（豁免词库）：添加免审核词汇，解决教学专用词误判
   - 分类管理：按政治 / 师德 / 广告等分类，单独开启 / 关闭某类风控

   

2. **风控规则配置**

   - 管控等级设置：单类违规选择「直接拦截」「弹窗提醒」「仅日志记录」
   - 场景策略：直播 / 课件 / 私聊 / 公告 分开配置不同严格规则
   - 时效管控：可设置临时风控（如节假日、招生季加强广告类审核）

   

3. **审核记录 & 日志中心**

   - 全量违规记录：时间、账号、内容、违规类型、处置结果、操作人员
   - 数据统计：违规 Top 词汇、高频违规教师、违规时段分析（用于校方整改）
   - 日志导出：支持 Excel/PDF 导出，满足监管备查要求

   

4. **人工复核中心**

   - AI 判定 “疑似违规” 的内容统一进入待复核列表
   - 管理员标记：确认违规 / 判定误判，结果反哺模型（持续优化）

   

### 模块 2：教师端（轻量化、易用为主，不打扰正常教学）

分**自查工具、实时提醒**两类形态：

1. **文本自查工具**（独立页面 / 插件）

   - 粘贴文字、上传 Word/PPT/TXT 文档，一键审核
   - 结果展示：标红违规内容、标注违规类型、给出修改建议

   

2. **实时场景提醒**

   - 直播 / 线上课堂：后台静默监测，**仅教师端弹窗提醒**（不打断课堂、不对外公示）
   - 师生私聊 / 班级群：发送前预检测，违规则提示 “内容包含违规话术，请修改”

   

3. **个人违规记录**

   - 教师仅查看自己的审核记录，支持申诉（申诉流转至管理员复核）

   

### 模块 3：实时风控中台（底层服务，对接各教学系统）

- 接口服务：提供 API，对接校内平台、直播课堂、家校群、备课软件
- 流量管控：高并发场景（全校直播课）保障响应速度
- 告警推送：严重违规（政治言论、恶意辱骂）实时推送消息给管理员

## 四、核心使用流程（全链路演示）

### 流程 1：老师课前自查课件

1. 老师上传 PPT / 教案 → 系统 OCR + 文本解析
2. AI 三层风控检测 → 标红违规内容 + 说明原因
3. 老师修改后重新提交 → 审核通过，正常使用

### 流程 2：课堂直播实时风控

1. 老师开启直播 → ASR 实时转文字，送入风控
2. 出现疑似违规话术 → 教师端**静默弹窗提醒**（学生无感知）
3. 严重违规 → 系统临时禁言 + 立即推送告警给管理员

### 流程 3：管理员后台运维

1. 每日查看违规报表、复核疑似内容
2. 将人工判定结果反馈给算法，迭代模型
3. 根据学校新规，更新自定义违禁词库

## 五、版本规划 & 落地节奏（产品迭代路线）

### V1.0 基础版（最小可行产品 MVP，快速上线）

- 核心：规则引擎 + 基础文本审核
- 功能：固定违禁词匹配、文本粘贴自查、基础词库管理、审核日志
- 适配场景：课件、书面文字审核

### V2.0 进阶版（补齐教育核心能力）

- 新增：NLP 谐音 / 变体识别、简易 LLM 语义识别、文档批量上传
- 优化：分级管控、白名单、违规分类统计

### V3.0 完整版（全场景覆盖）

- 新增：ASR 语音转写、OCR 图片审核、直播实时风控、API 对接
- 优化：大模型深度微调、申诉流程、数据看板、多角色权限

### V4.0 智能化迭代版

- 新增：违规趋势分析、风险预警（某班级 / 教师高频违规提前提醒）
- 模型自学习：人工复核数据自动训练，降低误判漏判

## 六、风险把控 & 产品红线（教育产品重中之重）

### 1. 误判风险（最大痛点）

- 方案：强制白名单、人工复核通道、模型持续迭代；教学专业词汇优先加入系统默认白名单
- 原则：**宁可多提醒，不错判处罚**，避免影响正常教学

### 2. 隐私风险（师生沟通内容敏感）

- 数据合规：仅用于风控，**严禁存储、泄露师生私人聊天、课堂隐私内容**
- 权限隔离：普通管理员看不到完整聊天原文，仅违规内容留痕；严格划分数据权限

### 3. 合规风险（紧跟教育政策）

- 词库定期更新：同步教育部、网信办最新师德规范、管控要求
- 留痕合规：所有审核记录长期留存，满足监管检查

### 4. 体验风险（不干扰教学）

- 课堂场景：只提醒老师，不打断授课、不公开公示
- 提醒方式：轻量弹窗，不强制阻断正常上课（严重违规除外）

## 七、运营 & 持续优化

1. **用户反馈通道**：老师可提交误判案例、新增违禁建议
2. **模型迭代**：每周汇总人工复核数据，迭代 NLP / 大模型
3. **词库运营**：建立行业词库共享机制，统一更新通用违规词
4. **使用培训**：给校方、老师讲解功能、违规类型，从源头减少违规

## 补充：产品 KPI 指标（衡量效果）

1. 功能指标：文本审核响应时间 < 500ms，语音 / 图片审核 < 2s
2. 效果指标：违规**漏检率 < 1%**，正常内容**误判率 < 2%**
3. 业务指标：教师自查使用率、疑似违规人工复核完成率
4. 合规指标：审核日志完整率、数据合规零泄露
